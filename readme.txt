== Description ==

Using Bitlnk, your posts are automatically shortened so you can easily share them through social networks, blogs and to family and friends. Bitlnk shortens your long blog URL's into short, easy to remember tiny links.

== Installation ==

1. Install the plugin either via the WordPress.org plugin directory, or by uploading the files to your server.
2. Activate the plugin through the plugins menu.
4. That's it. All of your new posts will be shortened in the right hand side of the page.
